# Indian Recipe Book 🍽️

A beautiful and modern recipe book application featuring authentic Indian food recipes with an attractive, responsive UI.

## Features

- 🍛 **8 Authentic Indian Recipes** - Including Butter Chicken, Biryani, Palak Paneer, Masala Dosa, and more
- 🔍 **Search Functionality** - Search recipes by name or description
- 🏷️ **Category Filtering** - Filter recipes by category (Main Course, Vegetarian, Breakfast, Snacks)
- 📱 **Responsive Design** - Works beautifully on desktop, tablet, and mobile devices
- 🎨 **Modern UI** - Attractive gradient colors, smooth animations, and clean design
- 📖 **Detailed Recipe Pages** - View complete ingredients and step-by-step instructions

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The production build will be in the `dist` folder.

## Recipes Included

1. **Butter Chicken** - Creamy tomato-based curry with tender chicken
2. **Biryani** - Fragrant basmati rice layered with spiced meat
3. **Palak Paneer** - Creamy spinach curry with soft paneer cubes
4. **Masala Dosa** - Crispy fermented crepe with spiced potato filling
5. **Chole Bhature** - Spicy chickpea curry with fluffy deep-fried bread
6. **Dal Makhani** - Creamy black lentils slow-cooked with butter
7. **Tandoori Chicken** - Yogurt-marinated chicken with smoky flavors
8. **Samosas** - Crispy triangular pastries with spiced potatoes
